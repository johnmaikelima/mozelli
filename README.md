# Mozelli Flooring & Tile - Landing Page

## Overview
Premium landing page focused on lead generation from Google Ads for Mozelli Flooring & Tile, a high-end flooring company serving Southwest Florida.

## Features
- **Interactive Before/After Slider** - Engaging hero section with drag-to-compare functionality
- **Mobile-First Responsive Design** - Optimized for all devices
- **Conversion-Optimized** - Multiple CTAs strategically placed throughout
- **Premium Design** - Dark/charcoal background with gold accents
- **Fast Loading** - Minimal dependencies, optimized performance

## Files Included
- `index.html` - Main landing page structure
- `styles.css` - Premium styling with dark theme and gold accents
- `script.js` - Interactive features (slider, mobile menu, form handling)
- `logo.png` - Company logo (replace with actual logo)

## Setup Instructions
1. Replace `logo.png` with the actual Mozelli Flooring & Tile logo
2. Open `index.html` in a web browser to view the landing page
3. The page is ready to deploy to any web hosting service

## Key Sections
1. **Header** - Sticky header with Request Quote and Call Now buttons
2. **Hero** - Interactive before/after slider with trust indicators
3. **Services** - Four main service categories with descriptions
4. **Why Mozelli** - Six key benefits highlighting company strengths
5. **Reviews** - Three customer testimonials with 5-star ratings
6. **Final CTA** - Strong conversion section
7. **Quote Form** - Lead capture form
8. **Footer** - Business information and hours
